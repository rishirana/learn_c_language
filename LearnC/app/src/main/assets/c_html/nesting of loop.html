Nesting of Loop

C programming allows to use one loop inside another loop. This is called Nesting of loops.

Syntax:

For Loop

for ( init; condition; increment )
{
  for ( init; condition; increment )
   {
     statement(s);
   }
 statement(s);
}

While Loop

while ( condition )
{
 while( condition )
 {
  statement(s);
 }
statement(s);

Do While Loop

do
{
statement(s);
 do
 {
  statement(s);
 }while( condition );
}while( condition );
