Functions

A function is block of statements, which is used to perform a specific task.

A c program has at least one function named main()

Types of functions:

There are two types of function in C

  1. Library function.
  2. User defined function

Library Function :

Library functions are the built-in function in C such as puts(),gets(),printf(),scanf() etc. These are the functions which already have a definition in the header files.

User defined function:

C allows programmer to define their own function according to their requirement.

Function Declaration

General sytax of function declaration is :

return_type function_name(parameter list);

Function definition

General sytax of function definition is :

return_type function_name (argument list)
{
Block of code
}

Explanation:

return_type : return type are the type of data that is returned by the function. If nothing is returned then the return type is void.

function_name : is the name of the function that is used when we call it from other function.

argument list : argument list contains variables names along with their data types. These arguments are inputs for the function.

Syntax :

...
void function_name()
{
// code
}
int main()
{
//calling function
function_name();
}

Advantages of user-defined function

✪ The program will be easier to understand, maintain and debug.

✪ Reusable codes that can be used in other programs.

✪ A large program can be divided into smaller modules. Hence a large project can be divided among many programmers.
