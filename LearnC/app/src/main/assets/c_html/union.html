Union

Unions are quite similar to structures in C. Like structures, unions are also derived types. The only difference is in storage. In structure each member has its own storage location , whereas all members of union uses a single shared memory location which is equal to largest data member.
Declaration:

Union variables can be created in similar manner as structure variables. It is declared using unionkeyword.

union car
{
char name[50];
int price;
};

Or
union car
{
char name[50];
int price;
};
int main()
{
union car car1, car2, *car3;
return 0;
}

Accessing member of union:

Again, the member of unions can be accessed in similar manner as structures.

In the above example, suppose you want to access price for union variable car1, it can be accessed as:

car1.price;

Likewise, if you want to access price for the union pointer variable car3, it can be accessed as:

(*car3).price
or
car3->price
