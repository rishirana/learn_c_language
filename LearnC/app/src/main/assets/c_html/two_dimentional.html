2D Arrays

It is collection of data elements of same data type arranged in rows and columns (that is ,in two dimensions).

Initialization

There are many ways to initialize 2D Arrays :
int disp[2][4] = { {10,11,12,13}, {14,15,16,17} };
Or
int disp[2][4] = { 10,11,12,13,14,15,16,17};

Things to Consider

You must remember that when we give values during one dimensional array declaration, we don't need to mention dimension. But that's no the case with 2D array. you must specify the second dimension even if you are giving values during the declaration . Let's understand this with the help of few examples
